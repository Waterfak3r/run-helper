# 跑步数据上传助手

## App 界面说明

打开 App 后可以看到以下输入项：

| 字段 | 说明 | 默认值 |
|------|------|--------|
| 学号 | 填入你的学号 | (无) |
| 学校代码 | 学校编号 | 10032 |
| 跑步距离 | 上传的跑步里程 (米) | 2345.0 |
| 配速 | 平均速度 (米/秒) | 3.3 |
| 用时 | 跑步总时长 (秒) | 648 |

填好学号后点击 **开始上传**，App 会自动依次执行：

1. 登录 —— 用学号 MD5 登录服务器
2. 获取数据 —— 查询当前跑步记录
3. 上传数据 —— 提交模拟跑步数据
4. 刷新数据 —— 确认上传结果

运行日志会实时显示在下方。

## 打包 APK (Windows)

### 第一步：安装 WSL

用管理员身份打开 PowerShell，执行：

```
wsl --install -d Ubuntu
```

装完重启电脑，打开 Ubuntu 终端。

### 第二步：安装 Buildozer

在 Ubuntu 终端中依次执行：

```bash
sudo apt update
sudo apt install -y python3-pip python3-dev git openjdk-17-jdk \
  zip unzip autoconf libtool pkg-config zlib1g-dev \
  libncurses5-dev libncursesw5-dev libtinfo5 cmake \
  libffi-dev libssl-dev

pip3 install --upgrade Cython buildozer
```

### 第三步：构建 APK

```bash
# 把项目复制到 WSL
cp -r /mnt/d/Programming/code/py/fuckhongqingting-master ~/fuckhqt-app
cd ~/fuckhqt-app

# 开始构建 (首次约 15-30 分钟)
buildozer android debug
```

### 第四步：取出 APK

```bash
cp ~/fuckhqt-app/bin/*.apk /mnt/d/
```

APK 会出现在 D 盘根目录，传到手机安装即可。

## 打包 APK (Linux / Mac)

如果你的系统本身就是 Linux 或 macOS，跳过 WSL 步骤，直接在项目目录下：

```bash
pip3 install --upgrade Cython buildozer
buildozer android debug
```

> macOS 上还需要安装 Java：`brew install openjdk@17`

## 常见问题

**Q: buildozer 报错 "command not found"**
A: 用 `python3 -m buildozer android debug` 或重启终端。

**Q: 手机安装后闪退**
A: 确保手机已允许安装"未知来源"应用，且 Android 版本不低于 5.0。

**Q: 点击开始后提示登录失败**
A: 检查手机是否在校园网环境下 (能访问 222.28.94.116)，以及学号和学校代码是否正确。
